# Droid-Server Interface (DSI)

This interface provides an API for managing droids on a server.

## Functionalities

- Create/Delete/Update droids
- Start/Stop/Restart droids
- Get droid logs
- Get droid status
- Schedule snoozing of droids


## Builders and Buildpacks

Builder Options:
- `heroku-18`
- `heroku-20`
- `heroku-20-build`
